# sim_radar

## 虚拟天气雷达 - 前端

项目介绍:

​	这个项目是我大学的毕业设计. 项目功能主要: 根据用户设置的雷达参数, 天气类型, 绘图设置等, 返回  雷达基本反射率dBZ和径向速度Vr.并在前端使用canvas绘制雷达观测的过程.可以帮助 学习气象的同学, 观察不同雷达对同一天气过程的影响.
